
# A simple GraphQL API for managing people photos

## Description

This project is made out of little joy.

I use Nest(NestJS) which is a framework for building Node.js server-side application

## Installation

```bash
npm ci
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
